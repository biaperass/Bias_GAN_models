# BFFHQ Preprocessing & Training Pipeline

This pipeline reproduces the BFFHQ bias-amplification experiments across multiple values of $\rho$ (label–bias correlation). Run the preprocessing script before launching training.

## Prerequisites

Before running the preprocessing or training pipeline, create and activate a Conda environment using one of the environment files provided in the repository root:

* `environment_cu12.yml` for systems with CUDA 12
* `environment_cu11.yml` for systems with CUDA 11

Choose the environment file that matches the CUDA version available on your machine or server.

## Step 1: Preprocessing

```bash
python bffhq_prep.py
```

This script runs the full preprocessing pipeline:

1. **Generate BFFHQ datasets** — creates BFFHQ datasets for the supported values of $\rho$
2. **Export images and metadata** — saves the processed images together with the corresponding metadata
3. **Write `dataset.json`** — generates the label files required by `dataset_tool.py`
4. **Run `dataset_tool.py`** — converts each image folder into the format expected by StyleGAN2, storing the output in:

```text
bffhq/data/bffhq_rho_gan/bffhq_{rho_tag}/
```

Each generated dataset is ready for StyleGAN2 training.

## Step 2: Training

Training is executed one $\rho$ value at a time using the provided shell wrapper. Each run trains a StyleGAN2 model on a single BFFHQ dataset.

```bash
bash bffhq_train.sh <rho_tag>
```

For example:

```bash
bash bffhq_train.sh 95
```

```bash
bash bffhq_train.sh 99
```

**Note:** for the BFFHQ dataset with $\rho = 0.995$, use `99` as the `rho_tag`.

The script automatically launches `train.py` using the dataset corresponding to the selected bias level.

Additional command-line arguments are forwarded directly to `train.py`, allowing hyperparameter overrides without modifying the shell script.

For example:

```bash
bash bffhq_train.sh 95 --gpus=2 --batch=64 --kimg=5000
```

## Dataset Requirements

The training script expects the preprocessing outputs generated by Step 1:

```text
bffhq/data/bffhq_rho_gan/bffhq_{rho_tag}
```

For example:

```text
bffhq/data/bffhq_rho_gan/bffhq_95
```

If the dataset directory does not exist, run preprocessing first:

```bash
python bffhq_prep.py
```

## Default Hyperparameters

| Flag | Default | Meaning |
|------|---------|---------|
| `--gpus` | 1 | Number of GPUs |
| `--batch` | 32 | Batch size |
| `--kimg` | 3500 | Training length (thousands of images) |
| `--cond` | 1 | Class-conditional training |
| `--mirror` | 1 | Horizontal flip augmentation |
| `--cfg` | stylegan2 | StyleGAN2 configuration |

## Optional Overrides

Any additional arguments passed to the training script are forwarded directly to `train.py`.

For example:

```bash
bash bffhq_train.sh 95 --batch=64 --gpus=2 --gamma=8.2
```

This allows overriding any supported StyleGAN2 training parameter without editing `bffhq_train.sh`.
